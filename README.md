# Mpox Skins Lesion Classification Deployment Model
Luciano Rizky Pratama
# Installation
Create Venv
```
python -m venv venv
```
Installing module
```
pip install -r requirements.txt
```
# Download the model 
Download and put it inside the folder
```
https://bit.ly/MpoxModel
```
# Run Project
Run with the command
```
streamlit run app.py
```
